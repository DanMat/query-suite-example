# Security Policy

This repository is a demonstration project. If you find a security issue in the
example itself, please report it **privately** rather than opening a public issue:

- Preferred: use GitHub's [**Report a vulnerability**](https://github.com/DanMat/query-suite-example/security/advisories/new)
  (Security tab → Advisories), or
- Email **dannymatthew@gmail.com**.

For vulnerabilities in the published libraries, please report them on the
corresponding package repository:
[query-fetch](https://github.com/DanMat/query-fetch/security),
[accept-query](https://github.com/DanMat/accept-query/security),
[query-cache](https://github.com/DanMat/query-cache/security),
[query-server](https://github.com/DanMat/query-server/security).
